# PRODIGY_DS_01

## Task 1: Population Distribution Visualization

### Objective
Create a histogram to visualize the distribution of population across countries using the World Bank Population Dataset.

### Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

### Dataset
World Bank Population Dataset

### Steps Performed
1. Loaded the dataset.
2. Checked for missing values.
3. Performed basic data cleaning.
4. Created a histogram for population distribution.
5. Analyzed the distribution pattern.

### Conclusion
The histogram shows that most countries have relatively smaller populations, while only a few countries have very large populations. This results in a right-skewed distribution of population across countries.
